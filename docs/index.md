# BALDR Accessibility Scan

## Outil open source d'audit d'accessibilité RGAA et WCAG enrichi par l'Intelligence Artificielle (IA)

BALDR Accessibility Scan est une solution open source d'**audit d'accessibilité web**, de **tests d'accessibilité automatisés** et de **conformité RGAA et WCAG**.

Conçu pour les équipes de développement, QA, accessibilité et DevOps, BALDR permet d'automatiser les contrôles d'accessibilité, d'intégrer les audits dans les pipelines CI/CD et de produire des rapports détaillés afin d'améliorer continuellement la conformité des applications web.

### Pourquoi BALDR ?

- Audit d'accessibilité automatisé
- Conformité RGAA et WCAG
- Analyse basée sur axe-core
- Navigation automatisée avec Puppeteer
- Prise en charge des espaces authentifiés
- Parcours utilisateurs multi-pages
- Intégration CI/CD
- API REST et CLI
- Rapports HTML, JSON et CSV
- Analyse enrichie par Intelligence Artificielle

**Contrairement aux scanners d'accessibilité traditionnels, BALDR analyse les parcours utilisateurs réels, y compris les applications authentifiées, les espaces clients, les intranets et les workflows métiers complexes.**
