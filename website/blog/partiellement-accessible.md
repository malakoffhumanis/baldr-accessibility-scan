Le site BALDR Accessibility Scan est actuellement **partiellement conforme**.

Cet audit d'accessibilite a ete genere automatiquement par l'outil BALDR, qui inclut de l'intelligence artificielle pour ses analyses.

Consulter le rapport detaille : [report-baldr.html (nouvelle fenêtre)](pathname:///report-baldr.html).

Nous travaillons en continu a l'amelioration de l'accessibilite numerique, notamment sur la conformite RGAA et l'ergonomie des parcours clefs.

Si vous rencontrez une difficulte d'acces a un contenu ou a une fonctionnalite, vous pouvez ouvrir une issue sur le depot GitHub du projet :

- https://github.com/malakoffhumanis/baldr-accessibility-scan/issues
